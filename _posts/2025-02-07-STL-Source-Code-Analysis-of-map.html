[STL] Source Code Analysis of map

2025-02-07

C/C++ STL SGI

map与set一样，也是使用STL的红黑树为底层实现，使用适配器适配出的一个容器，区别在于map的值类型是一个pair：

_Tp& operator[](const key_type& __k) {

// __i->first is greater than or equivalent to __k.

__i = insert(__i, value_type(__k, _Tp()));

return (*__i).second;

}
