可以看到，set实际上是使用了适配器模式，将红黑树适配为一个set，提供给用户使用。不过需要注意的是，STL的红黑树的插入提供了insert_unique和insert_equal两种方法，由于set中每个元素都是唯一的，因此set只能使用insert_unique`插入。
