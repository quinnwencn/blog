[C/C++] lambda函数原理初探

2024-08-05

在函数式编程中，lambda可以说是功能非常强大。举个例子，我们有一个数组，里面包含了某次购物时物品的价格，我们希望在结账时对这些价格进行求和，得出总共的价格，那么一开始的实现可能是这样的：

``C++

double printPrice(const std::vector& prices) {

double sum = 0;

for (auto d: prices) {

sum += d;

}

return sum;

}

``
