    这里的几个配置项的意义如下：

    * GPIO output level: GPIO初始化时的输出电平，可选High或者Low

    * GPIO mode: GPIO的模式，在Output模式下分为Output Push Pull和Output Open Drain，他们分别指：

    * Output Push Pull：芯片内部有两个MOSFET，上面的P-MOSFET连接到VDD，负责输出高电平（Push); 下面的 N-MOSFET链接到GND，负责拉低电平（Pull），因此可以设置为输出高电平还是低电平（吸收电流）

    * Output Open Drain: 芯片内部只有一个N-MOSFET，漏极接到GPIO引脚，源极接到GND，当写0时NMOS导通，引脚被拉低；当写1时NMOS截止，引脚是高阻态，什么也不驱动

    *

    * GPIO Pull-up/Pull-down: Pull-up接一个电阻，电阻另一端接VDD，Pull-down接一个电阻，电阻另一端接GND，用于解决悬空状态（高阻态）时的GPIO的默认电平

    * Maximum output speed: STM32的GPIO配置参数，控制电平转换速度

    * User Label：对配置的GPIO的别名

    void MX_GPIO_Init(void)

    {

    __HAL_RCC_GPIOB_CLK_ENABLE();

    /Configure GPIO pin Output Level /

    HAL_GPIO_WritePin(LED3_GPIO_Port, LED3_Pin, GPIO_PIN_RESET);
