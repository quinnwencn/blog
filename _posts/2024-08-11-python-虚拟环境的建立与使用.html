[python] 虚拟环境的建立与使用

2024-08-11

python
  • 安装虚拟环境

    • `` bash

    pip3 install virtualenv

    `

    或者

    `bash

    python3 -m pip install virtualenv

    `
  • 创建虚拟环境
    • `bash

    python -m venv path_of_your_venv

    `

    或者

    `bash

    virtualenv path_of_your_venv

    `
  • 激活虚拟环境
    • `bash

    source venv/bin/activate

    `

    激活后，termninal后显示venv：

  • 退出虚拟环境
    • `bash

    deactivate

    ``
